# Reliance Retail Customer Purchase Behaviour Analysis (2021-2025)

## Project Overview

This project focuses on analyzing customer purchasing behaviour using a five-year retail sales dataset. The objective is to identify sales trends, customer segments, payment preferences, and top-performing products to support data-driven business decisions.

The dashboard was developed using Power BI, and data analysis techniques were applied to generate meaningful business insights.
## Business Problem

Retail organizations generate large volumes of customer transaction data every day. Understanding customer purchasing patterns helps businesses improve customer retention, inventory planning, and sales performance.

This project addresses:

- Sales trend analysis
- Customer segmentation
- Payment preference analysis
- Product performance analysis
- Monthly purchase trends
- Geographical sales analysis
- Business KPI reporting

- Premium customers contribute significantly to total sales.
- Monthly trend analysis helps identify peak sales periods.
